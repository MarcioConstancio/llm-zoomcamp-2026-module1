# llm-zoomcamp-2026-module1

Activities from Module 1 of LLM ZOOMCAMP

For this activity I'm using gpt-4.1-mini from [Microsoft Foundry](https://azure.microsoft.com/en-us/products/ai-foundry). For this reason there is a base url in the `.env` file.

To replicate it in your machine, clone the project, rename `.env.example`to `.env` and add your keys to the file.
